# RACORP STORE

Website link-in-bio bertema cyberpunk merah. Seluruh tampilan berada dalam satu file `index.html`, sedangkan gambar tersimpan di folder `assets`.

## Membuka di komputer

Klik dua kali `index.html` untuk melihat website.

## Upload ke GitHub Pages

1. Buat repository GitHub baru, misalnya `racorp-store`.
2. Upload `index.html` dan folder `assets` ke bagian utama repository.
3. Buka **Settings → Pages**.
4. Pada **Build and deployment**, pilih **Deploy from a branch**.
5. Pilih branch **main**, folder **/(root)**, lalu klik **Save**.
6. Tunggu beberapa menit sampai alamat website muncul.

Jangan mengubah nama atau lokasi folder `assets`, karena `index.html` mengambil gambar dari folder tersebut.
